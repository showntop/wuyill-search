---
title: Wuyill als Ihre Suchmaschine einstellen
description: So stellen Sie Wuyill als Ihre standardmäßige Suchmaschine im Browser ein
---

<Steps>

### Öffnen Sie die Browsereinstellungen

Geben Sie die URL "chrome://settings/searchEngines" in die URL-/Navigationsleiste ein.

### Fügen Sie Wuyill als Suchmaschine hinzu

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine1.png" alt="wuyill zur Browsersuchmaschine hinzufügen 1" />

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine2.png" alt="wuyill zur Browsersuchmaschine hinzufügen 2" />

-   Der Name der Suchmaschine: Wuyill
-   Die Abkürzung: wuyill
-   Die URL mit %s anstelle der Abfrage: https://www.wuyill.com/search?q=%s

### Stellen Sie Wuyill als standardmäßige Suchmaschine ein

<img
    className="mt-4 overflow-hidden rounded-lg border"
    src="/images/doc/default-search-engine.png"
    alt="wuyill als Standard-Suchmaschine im Browser einstellen"
/>
</Steps>

Jetzt können Sie Wuyill direkt aus der Suchleiste Ihres Browsers verwenden.
