## So deployen Sie den Wuyill Vector Service auf Fly.io

<Steps>

### Fly.io-Konto erstellen (Optional)

-   Gehen Sie zu [Fly.io](https://fly.io) und erstellen Sie ein Konto.

### Flyctl installieren (Optional)

-   Installieren Sie `flyctl`, indem Sie die Anweisungen [hier](https://fly.io/docs/flyctl/install/) folgen.

### Wuyill-Code von GitHub klonen

```js
git clone https://github.com/showntop/wuyill.git
```

### Wuyill Vector Service deployen

```js
cd vector

cp fly.example.toml fly.toml

Ändern Sie den APP_NAME in fly.toml in Ihren App-Namen

fly deploy
```

</Steps>
