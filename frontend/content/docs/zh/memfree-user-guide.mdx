## MemFree 搜索

### AI模型的使用案例

-   **GPT-4o Mini**：流式响应。适用于轻量级任务，如文本摘要、翻译、从图像中提取文本等。

-   **GPT-4o**：流式响应。适用于最复杂的任务，如复杂写作任务、高级内容创作和研究辅助等。

-   **O1-Mini**：慢速。推理模型。特别擅长编码、数学和科学。

-   **O1-Preview**：慢速。推理模型。O1-Mini的升级版，能够解决跨领域的复杂问题。

-   **Claude 3.5 Sonnet**：流式响应。与O1-Mini在编码方面同样出色。
