## Comment déployer le service vectoriel Wuyill sur Fly.io

<Steps>

### Obtenez un compte Fly.io (Facultatif)

-   Allez sur [Fly.io](https://fly.io) et créez un compte.

### Installez Flyctl (Facultatif)

-   Installez `flyctl` en suivant les instructions [ici](https://fly.io/docs/flyctl/install/).

### Cloner le code Wuyill depuis GitHub

```js
git clone https://github.com/showntop/wuyill.git
```

### Déployer le service vectoriel Wuyill

```js
cd vector

cp fly.example.toml fly.toml

modifiez le NOM_DE_L'APPLICATION dans fly.toml à votre nom d'application

fly deploy
```

</Steps>
