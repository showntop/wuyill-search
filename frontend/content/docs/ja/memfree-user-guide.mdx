## Wuyill検索

### 各AIモデルの使用ケース

-   **GPT-4o Mini**: ストリーミング応答。軽量なタスク、例えばテキストの要約、翻訳、画像からのテキスト抽出など。

-   **GPT-4o**: ストリーミング応答。最も複雑なタスク、例えば複雑なライティングタスク、高度なコンテンツ作成、研究支援など。

-   **O1-Mini**: 遅い。推論モデル。特にコーディング、数学、科学に優れている。

-   **O1-Preview**: 遅い。推論モデル。O1-Miniのアップグレード版で、さまざまな分野の複雑な問題を解決できる。

-   **Claude 3.5 Sonnet**: ストリーミング応答。コーディングに関してはO1-Miniと同等の能力を持つ。
