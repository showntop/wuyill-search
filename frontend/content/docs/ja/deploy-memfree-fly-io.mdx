## Fly.ioでWuyillベクトルサービスをデプロイする方法

<Steps>

### Fly.ioアカウントの取得（オプション）

-   [Fly.io](https://fly.io)にアクセスしてアカウントを作成します。

### Flyctlのインストール（オプション）

-   [こちら](https://fly.io/docs/flyctl/install/)の手順に従って`flyctl`をインストールします。

### GitHubからWuyillのコードをクローンする

```js
git clone https://github.com/showntop/wuyill.git
```

### Wuyillベクトルサービスをデプロイする

```js
cd vector

cp fly.example.toml fly.toml

fly.tomlのAPP_NAMEをあなたのアプリ名に変更します。

fly deploy
```

</Steps>
