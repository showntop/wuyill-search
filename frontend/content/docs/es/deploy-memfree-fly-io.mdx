## Cómo desplegar el servicio Vector de Wuyill en Fly.io

<Steps>

### Obtener una cuenta de Fly.io (Opcional)

-   Ve a [Fly.io](https://fly.io) y crea una cuenta.

### Instalar Flyctl (Opcional)

-   Instala `flyctl` siguiendo las instrucciones [aquí](https://fly.io/docs/flyctl/install/).

### Clonar el código de Wuyill desde GitHub

```js
git clone https://github.com/showntop/wuyill.git
```

### Desplegar el servicio Vector de Wuyill

```js
cd vector

cp fly.example.toml fly.toml

cambia el APP_NAME en fly.toml por el nombre de tu aplicación

fly deploy
```

</Steps>
