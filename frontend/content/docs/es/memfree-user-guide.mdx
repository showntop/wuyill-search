## Búsqueda de Wuyill

### Casos de uso para cada modelo de IA

-   **GPT-4o Mini**: Respuesta en streaming. Tareas ligeras, como resumir texto, traducción, extracción de texto de imágenes, etc.

-   **GPT-4o**: Respuesta en streaming. Tareas más complejas, como tareas de escritura complejas, creación de contenido avanzado y asistencia en investigación, etc.

-   **O1-Mini**: Lento. Modelo de razonamiento. Especialmente bueno en codificación, matemáticas y ciencias.

-   **O1-Preview**: Lento. Modelo de razonamiento. La versión mejorada de O1-Mini, puede resolver problemas complejos en diferentes dominios.

-   **Claude 3.5 Sonnet**: Respuesta en streaming. Tan bueno en codificación como O1-Mini.
